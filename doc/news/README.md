## Упоминание в СМИ

Получить список источников

**Ограничения по лицензии** :

Метод доступен только в расширенной лицензии

**URL** : `/news/source`

**Необязательные параметры** :
- `limit(str)` - количество записей в запросе, по умолчанию 10, максимум 100
- `page(str)` - страница запроса, по умолчанию 0

**Метод** : `GET`

**Формат ответа**

```json
{
  "type": "object",
  "properties": {
    "source_id": {
      "description": "идентификатор источника и его название",
      "type": "string"
    }
  }
}
```

**Пример запроса**

```text
https://api.sbis.ru/vok/news/source
```


**Пример ответа**

```json
{
  "1": "Lenta.ru",
  "7": "Известия.Ru",
  "6": "Московский Комсомолец (МК)",
  "8": "РИА Новости",
  "9": "ТАСС"
}
```

## Статистика по источникам

Получить статистику по новостям от источников

**Ограничения по лицензии** :

Метод доступен только в расширенной лицензии

**URL** : `/news/stat`

**Обязательные параметры** :
- `inn(str) or ogrn(str)` - ИНН или ОГРН контрагента

**Необязательные параметры** :
- `kpp(str)` - КПП контрагента

**Метод** : `GET`

**Формат ответа**

```json
{
  "type": "object",
  "properties": {
    "source_name": {
      "description": "название источника и количество новосте с него",
      "type": "int"
    }
  }
}
```

**Пример запроса**

```text
https://api.sbis.ru/vok/news/stat?inn=7736050003
```

**Пример ответа**

```json
{
  "Комсомольская правда": 3,
  "РБК (РосБизнесКонсалтинг)": 4,
  "Газета.ru": 19,
  "Интерфакс": 66
}
```

## Данные по упоминанию в СМИ

Получить записи по упоминанию в СМИ

**Ограничения по лицензии** :

Метод доступен только в расширенной лицензии


**URL** : `/news/data`

**Обязательные параметры** :
- `inn(str) or ogrn(str)` - ИНН или ОГРН контрагента

**Необязательные параметры** :
- `kpp(str)` - КПП контрагента
- `limit(str)` - количество записей в запросе, по умолчанию 10, максимум 100
- `page(str)` - страница запроса, по умолчанию 0
- `source([int])` - идентификаторы источников

**Метод** : `GET`

**Пример запроса**

```text
https://api.sbis.ru/vok/news/data?inn=7712040126&source=1&source=6&limit=55&page=1
```
